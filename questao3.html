Qual é a diferença entre linguagem de programação Server Side e Client Side?

Server Side

o codigo e processado no servidor antes de mandar

Client side

são executadas diretamente no navegador do usuário

Exemplo

Server Side

  • Validação de dados no servidor
  • Geração dinâmica de páginas web

Exemplos de Client Side

  • Animações Usando CSS
  • Manipulação de DOM com JavaScript
  • Formulários Dinâmicos com Validação
